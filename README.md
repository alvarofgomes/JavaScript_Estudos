### README — Curso Web Moderno Completo com JavaScript + Projetos

## Visão geral
Este repositório será usado para acompanhar meus estudos no curso **Curso Web Moderno Completo com JavaScript + Projetos**.
A ideia é registrar anotações, exercícios e pequenos projetos conforme eu for avançando.

## Objetivo
Construir uma base sólida para desenvolvimento **Web moderno**, evoluindo do **fundamental ao avançado**, passando por frontend e backend com JavaScript, além de bancos de dados e frameworks populares.

## O que vou aprender (resumo)
- Construir aplicações modernas para Web
- Criar aplicações reais usando tecnologias atuais do mercado
- **HTML5** (recursos mais recentes)
- **CSS3** (Flexbox, CSS Grid e boas práticas de estilo)
- **JavaScript de verdade**: sintaxe + raciocínio (como pensar em JS)
- Recursos do JavaScript moderno (**ES6+ / ES2015+**)
- Bibliotecas e frameworks: **React**, **Vue**, **Bootstrap**, **jQuery**
- Paradigmas: **Programação Funcional** e **Orientação a Objetos**
- Boas práticas e padrões de projeto
- Aplicações baseadas em componentes
- Projetos práticos (aplicações reais)
- Caminho para **Fullstack** (frontend + backend)
- Backend com **Node.js**
- Autenticação de usuário
- Bancos de dados: **MySQL**, **Postgres**, **MongoDB (NoSQL)**
- Acesso a bancos usando JavaScript/Node
- Builds profissionais com **Gulp** e **Webpack**
- Angular 9 (componentes, diretivas, pipes, services etc.)
